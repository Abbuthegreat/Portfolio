# Abbas Damaniya — Robotics Engineer Portfolio

A personal portfolio website built with React, Vite, TypeScript, and Three.js.

## 🚀 Tech Stack

- **React + Vite + TypeScript**
- **Three.js / React Three Fiber** — 3D robot models
- **Framer Motion** — animations and transitions
- **Tailwind CSS** — styling
- **shadcn/ui** — UI components

## 🤖 Features

- Interactive 3D robots (BB8 + custom CrazyRobot)
- Neon cursor effect with comet particle trail
- Scroll progress bar
- Skills ticker marquee
- Fully responsive — works on all screen sizes
- WhatsApp/LinkedIn link preview with custom OG image

## 📦 Run Locally

```bash
npm install
npm run dev
```

## 🌐 Deployed on Vercel

Auto-deploys on every `git push` to main branch.
